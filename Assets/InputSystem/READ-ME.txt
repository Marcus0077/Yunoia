    GAME CONTROLS:

    Exit Game = Escape (K&M) / Select (Xbox Controller)

    Movement Controls:

    - Walk = WASD (K&M) / Left Stick (Xbox Controller)
    - Jump = Spacebar (K&M) / A (Xbox Controller)
    - Dash = Shift (K&M) / B (Xbox Controller)

    Clone Controls:
    
        - Summon Clone = F (K&M) / Left Bumper (Xbox Controller)
        - Exit Clone = F (K&M) / Left Bumper (Xbox Controller)
        - Switch Between Clone & Player = R (K&M) / Right Bumper (Xbox Controller)

    Grapple Controls:

        - Shoot Hook = Left-Click (K&M) / Right Trigger (Xbox Controller)
        - Cancel Hook = Spacebar (K&M) / A (Xbox Controller)
        - Reel-In Hook = Left-Click (K&M) / Right Trigger (Xbox Controller)
        
    Push Controls:
    
        - Activate Push = Q (K&M) / Left Trigger (Xbox Controller)
        - Charged Push = Hold Q (K&M) / Hold Left Trigger (Xbox Controller)
        
    Pause Menu Controls:
        
        - Pause / Unpause game = P (K&M) / Start (Xbox Controller)

    Interaction Controls:
        
        - Interact With Object = E (K&M) / X (Xbox Controller)
