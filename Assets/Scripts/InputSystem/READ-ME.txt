*TEMPORARY* GAME CONTROLS:

    Exit Game = Escape (Keyboard)

    Movement Controls:

    - Walk = WASD (Keyboard) / Left Stick (Xbox Controller)
    - Jump = Spacebar (Keyboard) / A (Xbox Controller)
    - Dash = Shift (Keyboard) / Right Stick Press (Xbox Controller)

    Clone Controls:
    
        - Summon Clone = F (Mouse) / X (Xbox Controller)
        - Exit Clone = G (Keyboard) / B (Xbox Controller)
        - Switch Between Clone & Player = R (Mouse) / Y (Xbox Controller)

    Grapple Controls:

        - Shoot Hook = E (Keyboard) / Left Bumper (Xbox Controller)
        - Cancel Hook = Spacebar (Keyboard) / A (Xbox Controller)
        - Extend Hook = Control (Keyboard) / Right Bumper (Xbox Controller)
        
    Push Controls:
    
        - Activate Push = Q (Keyboard) / Left / Right Trigger (Xbox Controller)
        - Charged Push = Hold Q (Keyboard) / Hold Left / Right Trigger (Xbox Controller)
        
    Pause Menu Controls:
        
        - Pause / Unpause game = P (Keyboard) / Start (Xbox Controller)
