*TEMPORARY* MOVEMENT AND CLONE SUMMON PROTOTYPE CONTROLS:

    Movement Controls:

    - Walk = WASD (Keyboard) / Left Stick (Xbox Controller)
    
    - Jump = Spacebar (Keyboard) / A (Xbox Controller)

    Summon Controls:
    
        Custom Summon Mode:
        
        - Activate Summon Cursor = F (Keyboard) / Left Bumper (Xbox Controller)
        - Deactivate Summon Cursor = Escape (Keyboard) / B (Xbox Controller)
        - Move Summon Cursor = Mouse Position (Mouse) / Right Stick Position (Xbox Controller) *** Cursor starts at mouse position 
                                                                                                   regardless of if you are using controller; 
                                                                                                   If mouse is positioned outside of the play
                                                                                                   area, you may need to mouse the cursor down
                                                                                                   on controller so see it. ***
        - Summon Clone at Cursor Position = Left-Click (Mouse) / X (Xbox Controller)
        
        Preset Summon Mode: 
        
        - Summon Clone on Pressure Plate = Left-Click (Mouse) / X (Xbox Controller)
        
    Clone-Specific Controls:
    
        Exit Clone = G (Keyboard) / Y (Xbox Controller)
        Switch Between Clone & Player = Right-Click (Mouse) / Right Bumper (Xbox Controller)
